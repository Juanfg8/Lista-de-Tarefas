# Lista-de-Tarefas
C# POO Project - To-Do List Console App.  Aplicação de console em C# que explora Orientação a Objetos. O código demonstra: Tipagem estática, uso de classes (Tarefa, Gerenciador), propriedades com encapsulamento e manipulação de coleções de dados (List&lt;T>). Preparado para a apresentação V2.
